6-DOF Manipulator Impedance Control

基于 ROS 与 Gazebo 构建 6 自由度机械臂的物理仿真系统。实现基于运动学逆解的笛卡尔空间轨迹精确跟踪，并自研阻抗控制（Impedance Control）算法框架，实现机械臂末端与未知环境的柔顺接触与力控交互。 Built a physical simulation system for a 6-DOF robotic arm based on ROS and Gazebo. Implemented precise Cartesian trajectory tracking based on inverse kinematics, and developed a custom Impedance Control algorithm framework to achieve compliant contact and force-controlled interaction between the end-effector and unknown environments.

Vision-Guided Autonomous Bin Picking

结合 RGB-D 深度相机实现的工业级无序抓取仿真系统。利用 PCL 处理点云数据进行目标位姿估计，并深度定制 MoveIt2 的规划管线，通过 RRT* 算法实现复杂动态障碍物环境下的无碰撞运动规划。 An industrial-grade bin-picking simulation system integrated with RGB-D depth cameras. Utilized PCL to process point cloud data for target pose estimation, and deeply customized the MoveIt2 planning pipeline to achieve collision-free motion planning in complex dynamic obstacle environments using the RRT* algorithm.

Future Research · Embodied AI

Sim2Real: RL-based Robotic Arm Manipulation

探索具身智能的前沿落地。计划在 NVIDIA Isaac Sim 中搭建支持 GPU 加速的高度逼真强化学习环境，使用 PyTorch 训练 PPO 算法，让机械臂自主学习复杂的抓取、堆叠等非结构化任务，攻克 Sim2Real 域间隙难题。 Exploring the frontier deployment of Embodied AI. Planning to build a highly realistic, GPU-accelerated reinforcement learning environment in NVIDIA Isaac Sim. Will train the PPO algorithm using PyTorch to enable the robotic arm to autonomously learn complex unstructured tasks like grasping and stacking, tackling the Sim2Real domain gap challenge.
